2. HELPER IDEA: Does requested booking overlap an existing one? 

Assume each booking is 2 hours (120 mins) 

FUNCTION overlaps(requestedTime, existingTime): 
    reqStart = timeToMinutes(requestedTime) 
    reqEnd = reqStart + 120 

    exStart = timeToMinutes(existingTime)
    exEnd = exStart + 120 

    IF reqStart < exEnd AND reqEnd > exStart: 
        RETURN true 
    ELSE: 
        RETURN false 

This condition is the OVERLAP RULE
